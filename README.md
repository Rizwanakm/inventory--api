# Inventory Management API

A Node.js + TypeScript backend for managing inventory, orders, and stock tracking.

## Features
- Add, view, and update products
- Checkout and order management
- Auto stock update after payment
- Cancel order and restock if not shipped
- Swagger API documentation

## Tech Stack
- Node.js
- Express.js
- MongoDB
- TypeScript
- Swagger UI

## Setup
1. Clone the repo
2. Run `npm install`
3. Run `npm run dev` for local development
4. Run `npm run build && npm start` for production
